05 · Cruzamento de Dados
O que Meta Ads + Instagram + GA4 dizem juntos
Correlações entre os canais — insights que só aparecem cruzando as fontes
Cruzamento 1 — Ads ligam, orgânico não existe
Quando os ads estavam ativos (01–03/mai): reach IG ~17K/dia, sessões no site ~13/dia, +12 seguidores/dia. Quando pausaram (04–14/mai): reach ~50/dia, sessões no site ~0,5/dia, +0,3 seguidores/dia. A correlação é 1:1 — a Border só existe digitalmente quando está pagando para existir. Isso é o diagnóstico principal do mês.
Cruzamento 2 — Qual criativo de ads gera mais resultado real
Post impulsionado "design autoral brasileiro": CPC R$ 0,15, CTR 2,97%. Post "Se você não se vê nas...": CPC R$ 1,20, CTR 2,05%. Conteúdo de produto direto (estética + peça) converte 8x melhor em custo que conteúdo identitário emocional. Isso não significa abandonar o storytelling — significa separar os objetivos: branding orgânico usa a voz, conversão paga usa o produto.
Cruzamento 3 — O orgânico que funciona: "slow days"
O único reel orgânico com engajamento real (26 interações, 1 share, reach 176) foi o "slow days • border;crew" — estética de produto + cultura + sem copy de venda. É o formato que mais se aproxima do que os algoritmos distribuem organicamente. É também o mais difícil de escalar — mas é o que constrói base real.
Cruzamento 4 — O pixel de vendas está mentindo ou quebrado
Campanha 2 (objetivo vendas): R$ 61,86 gastos, 0 add-to-carts. Mas o GA4 mostra sessões chegando via ig/paid e fb/paid nesses dias. Alguém chegou no site — mas o pixel não registrou nenhuma ação. Ou o pixel de add-to-cart não está disparando corretamente, ou os usuários estão saindo antes de interagir com produtos. Isso precisa ser auditado urgente antes de qualquer novo gasto com objetivo de conversão.
A equação do momento
Inverno 2026 é o melhor momento de produto para a Border (alfaiataria desconstruída dominando editorial). Meta Ads estão 11 dias pausados. O orgânico sozinho não sustenta nem 100 contas alcançadas por dia. O pixel de conversão provavelmente está quebrado. Cada dia parado é um dia de janela de temporada perdido — mas religar os ads com o pixel quebrado é dinheiro no ralo. A sequência correta: corrigir pixel → religar Campanha 1 (reconhecimento) com criativo de produto → Campanha 2 apenas depois de confirmar o pixel funcionando.
